───────────────────────────────

- Print a markdown-style data quality verdict

SECTION 3: Price Behavior
──────────────────────────
- Adjusted close over time: subplot grid (2 columns, ceil(n_tickers/2) rows), one line per ticker
- Normalized price chart: rebase all tickers to 100 at DATE_START, overlay on one chart

  → create one combined chart showing close + both MAs

SECTION 4: Return Distribution Analysis
────────────────────────────────────────
- Summary stats table (pandas DataFrame): mean, std, skew, kurtosis, min, max 
  for simple_return AND log_return per ticker
- Histogram of daily simple_returns per ticker (subplots, 2 columns, KDE overlay)
- Q-Q plot vs normal distribution per ticker (use scipy.stats.probplot, subplots)
- Combined boxplot: simple_returns across all tickers side-by-side
- Shapiro-Wilk normality test per ticker → print p-values in a table
- Print markdown-style comment: which tickers violate normality? 
  What does this mean for Sharpe / Monte Carlo?

SECTION 5: Volatility Analysis
───────────────────────────────
- Rolling 30-day std of simple_return per ticker, annualized (multiply by sqrt(252))
- Line chart: rolling volatility over time (all tickers, separate lines)
- Heatmap: rows = month (YYYY-MM format), cols = ticker, values = monthly std of returns
- Identify and print: highest-volatility periods — what market events?
- Markdown-style comment: volatility regimes observed

SECTION 6: Correlation & Co-movement
──────────────────────────────────────
- Pivot returns to wide format (date × ticker) — use this for correlation only
- Pearson correlation matrix: compute and plot heatmap (annotated, diverging colormap)
- Spearman correlation matrix: compute and plot heatmap (same style)
- Identify top 3 most correlated pairs and bottom 3 (least / most negative)
- For those 6 pairs: create 6 scatter plots with regression line (numpy polyfit)
- Rolling 60-day Pearson correlation for top-correlated pair — plot over time
- Markdown-style comment: diversification verdict

SECTION 7: Sector & Metadata Analysis
──────────────────────────────────────
- Join returns.parquet with metadata.json on ticker → enrich with sector, industry, marketCap, beta
- Group by sector: compute mean return, std, sharpe ratio (assume risk-free rate = 0)
- Bar chart: sector-level average daily return (sorted)
- Bar chart: sector-level annualized volatility (sorted)
- Scatter plot: beta (from metadata) vs realized volatility (from data) → does beta predict vol?
- Print markdown-style comment: which sectors led / lagged?

SECTION 8: Outlier & Anomaly Detection
───────────────────────────────────────
- Top 10 largest positive and negative single-day moves per ticker → print as table
- Z-score of daily simple_return: count |z| > 3 per ticker, print results
- Flag and count zero-volume trading days (if any)
- Flag and count days with zero price change (close == open and volume > 0)
- Load cleaning_report.json and compare outlier counts from Step 2
- Markdown-style comment: did Step 2 catch what we see? Anything to revisit?

SECTION 9: Drawdown Preview
─────────────────────────────
- Cumulative returns per ticker: (1 + simple_return).cumprod() grouped by ticker
- Drawdown per ticker: (cum_return / cum_return.cummax()) - 1
- Line chart of drawdowns over time (all tickers, separate lines)
- Max drawdown and date of max drawdown per ticker → print as table
- Recovery time (days from max drawdown to recovery) per ticker
- Markdown-style comment: worst drawdown? Recovery patterns?
